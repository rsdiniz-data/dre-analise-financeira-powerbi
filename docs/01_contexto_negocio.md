# 1. Contexto de Negócio

A análise da Demonstração do Resultado do Exercício (DRE) é fundamental para avaliação da performance financeira de uma empresa.

No cenário analisado, os dados estavam disponíveis em arquivos não estruturados (PDF e Excel), o que gerava:

- Alto esforço manual
- Risco de erros
- Baixa padronização
- Dificuldade de análise histórica

Este projeto surge com o objetivo de estruturar esses dados e permitir análises consistentes e escaláveis.
