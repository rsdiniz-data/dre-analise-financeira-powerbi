# 1. Contexto de Negócio

A análise da Demonstração do Resultado do Exercício (DRE) é essencial para avaliar a performance financeira e apoiar a tomada de decisão.

No cenário analisado, os dados estavam disponíveis em arquivos não estruturados (PDF e Excel), o que gerava:

- Alto esforço manual no tratamento das informações  
- Maior risco de inconsistências e erros  
- Baixa padronização dos dados  
- Dificuldade para análises comparativas e históricas  

Diante desse contexto, o projeto foi desenvolvido com o objetivo de estruturar, padronizar e integrar esses dados em um modelo analítico, permitindo análises confiáveis, escaláveis e orientadas ao negócio.
