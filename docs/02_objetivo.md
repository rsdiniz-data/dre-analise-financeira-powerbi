# 2. Objetivo do Projeto

Desenvolver uma solução de Business Intelligence para automatizar a construção e análise da DRE, por meio de um pipeline de dados estruturado e um modelo analítico escalável.

A solução permite:

- Automatizar o processo de ingestão, tratamento e consolidação dos dados  
- Padronizar a estrutura das informações financeiras  
- Viabilizar análises históricas e comparativas  
- Gerar indicadores financeiros confiáveis (KPIs)  
- Apoiar a tomada de decisão com base em dados  

O objetivo final é transformar dados não estruturados em uma base analítica consistente, pronta para consumo em dashboards e análises estratégicas.
