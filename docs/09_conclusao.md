# 9. Conclusão

O projeto demonstra como técnicas de Business Intelligence e Engenharia de Dados podem transformar dados brutos em informação estratégica para tomada de decisão.

A solução evolui de um processo manual baseado em arquivos para um pipeline estruturado, com:

- Ingestão e padronização de dados
- Modelagem analítica (Star Schema)
- Cálculo de KPIs financeiros com DAX
- Camada analítica pronta para consumo em dashboards

Além disso, a implementação permite:

- Escalabilidade da solução
- Maior governança dos dados
- Facilidade de manutenção e evolução

Esse projeto reforça a importância de integrar dados, modelagem e análise para gerar valor real ao negócio.
